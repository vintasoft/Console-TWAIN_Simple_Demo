File "VintasoftTwainService.exe" is necessary in .NET Framework 4 project only
if you want to use 32-bit TWAIN drivers in 64-bit application.

Files "VintasoftTwainService.exe", "VintasoftTwainService.dll" and "VintasoftTwainService.runtimeconfig.json" are necessary
in .NET 5 or .NET Core 3 project only if you want to use 32-bit TWAIN drivers in 64-bit application.

More info about file VintasoftTwainService.exe please read here:
https://www.vintasoft.com/docs/vstwain-dotnet/Programming-Twain-How_To_Use_32bit_TWAIN_driver_in_64bit_Application.html
